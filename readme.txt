＊紹介

InternetExplorer用のコンテキストメニュー拡張機能はよく見かけますが、
Mozilla系ブラウザ用のものは見かけなかったので、作ってみました。
Mozilla Firefoxに以下のような機能を加えます。

・ユーザー定義のスタイルシートをページに適用する
・ユーザー定義のJavaScript（カスタムスクリプト）をメニューから実行する
・外部アプリケーションで現在のページを開く
・文法チェッカやインターネットアーカイブなどに現在のページを送る
・検索エンジンなどに選択範囲のテキストを送る
・選択範囲のテキストをURIとして開く
・コンテキストメニューに「ブックマーク」メニューを加える
・次の見出し/前の見出し/一つ上のディレクトリへ移動する
・ページのアウトラインを表示する
・ナビゲーション用リンクを一覧表示する
・標準の機能よりも使いやすいスタイル切り替え機能
・サイトごとにユーザースタイルを設定する
・リンクの一覧をコピーする
・コメント・要素のid/name・注釈・イベントといった不可視情報を表示する
・引用部の引用元を表示する
・追加/削除部分の説明を表示する
・画像の長い説明を表示する
・ソース表示のビューワーやメーラーを好みのものに変更する
・「見出しジャンプ」「フォーカスの移動」「リンク一覧のコピー」に
  S/W/A(Ctrl+↓)/Q(Ctrl+↑)/Ctrl+JのOpera ver.6互換のキーボードショート
  カットを割り当てる


========================================================================
＊インストール手順

  +-----------------------------------------------------------------+
  | ※インストールを行う前に、Preferences（設定） >                 |
  |   Advanced（詳細） > Software Installation（ソフトウェアのイ    |
  |   ンストール）で「Enable software installation（ソフトウェア    |
  |   のインストールを可能にする）」にチェックを入れて下さい。      |
  +-----------------------------------------------------------------+

step1: ファイルのコピー
  ctxextensions.xpiをブラウザのウィンドウにドロップすると、インストールが
  開始されます。

step2: Firefoxの再起動
  ファイルのコピーが終わったら、Firefoxを再起動します。
  コピーしたファイル群が、起動時にFirefoxに登録されます。
